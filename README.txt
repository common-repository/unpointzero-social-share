UnPointZero Social Share is a simple 100% customizable social sharing plugin without any external service that could slows down your website.

== Description ==

This plugin provides the ability to your visitors to share your website content on social networks.
It's 100% customizable (custom icons, buttons position...) and it doesn't use any external service that could slow down your website.

It's an early release, feel free to contact us is you found any bug or if anything is missing.

== Installation ==

How to install

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'UnPointZero Social Share'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. Go on Settings => Social share options to setup your plugin
6. You can use the shortcode [upzsocial] to display the buttons.

== Frequently Asked Questions ==

= How to use my own icons ? =
*It's pretty much easy. Create a folder on your actual WordPress theme directory and put your social icons images in it. Images must be named like this:
Facebook => facebook.png, Digg => digg.png, Google+ => google.png, Linkedin => linkedin.png, Pinterest => pinterest.png, Twitter => twitter.png
